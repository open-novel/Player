## ～このアプリケーションの利用について～

『Player』レポジトリ内のファイルはパブリックドメイン（CC0）で、商用非商用問わず改変・再頒布・転用が可能です。  
『Products』レポジトリ内のライセンスは、各作品フォルダ毎に違います。  
作品内にライセンスファイルがない場合は、著作権は手放していないがライセンス表記不要で、商用非商用問わず改変・再頒布・転用が可能です。
