～このアプリケーションの利用について～

	『エンジン』内のファイルはパブリックドメイン（CC0）で、商用非商用問わず改変や再頒布、転用が可能です。
	『作品』内のライセンスは、各作品フォルダ毎に違います。
	作品内にライセンスファイルがない場合はパブリックドメイン（CC0）です。
